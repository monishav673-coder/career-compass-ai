# 🎓 Career Compass AI

## 📌 Project Overview
Career Compass AI is a web-based career guidance system developed to help students discover suitable career paths based on their interests and academic preferences. The system provides personalized career recommendations through a simple and user-friendly interface.

## ✨ Features
- User Registration and Login
- Career Assessment
- Career Recommendations
- Career Details
- Career Roadmap
- User Profile
- Responsive Web Interface

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
- Local Storage

## 🚀 How to Run the Project
1. Download or clone this repository.
2. Open the project folder.
3. Open `register.html` to create a new account.
4. Log in using `login.html`.
5. Complete the career assessment and explore the recommendations.

## 📂 Project Structure
- `index.html` – Career Assessment
- `register.html` – User Registration
- `login.html` – User Login
- `profile.html` – User Profile
- `career-details.html` – Career Information
- `roadmap.html` – Career Roadmap

## 🎯 Project Objective
To provide students with an easy-to-use platform that recommends suitable career paths based on their interests and educational background.
